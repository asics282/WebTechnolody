Он же Питон, он же Пайтон

Это высокоуровневый язык программирования общего назначения с динамической строгой типизацией и автоматическим управлением памятью, ориентированный на повышение производительности разработчика, читаемости кода и его качества, а также на обеспечение переносимости написанных на нём программ

Гвидо Ван Россум - создатель языка программирования Питон

Пример кода:

name = input()
if name == 'CSS':
    print("Красиво")
elif name == 'HTML':
    print("Топорно")
else:
    print("Вам надо на курс по веб-технологиям")

Плюсы:
  • простота синтаксиса и визуальное восприятия кода
  • легкий и быстрый старт
  • изобилие библиотек и фреймворков
  • кроссплотформенность
  • поддержка ООП
  • инамическая типизация
Минусы:
  1. Низкая скорость
  2. Не подходит для задач, которые требуют большого объема памяти
